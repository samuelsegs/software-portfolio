# Mensajes-UnidadesTG

Sistema de automatización desarrollado en Python para el envío de mensajes 
informativos a unidades operativas mediante WhatsApp Web.

## Descripción
El proyecto automatiza la comunicación operativa a partir de información
contenida en archivos de viaje, reduciendo tiempos de notificación y errores
manuales en procesos repetitivos.

## Objetivo
Optimizar la comunicación con unidades mediante un sistema automatizado,
mejorando la eficiencia operativa y la distribución de información.

## Tecnologías utilizadas
- Python
- Manejo de archivos (CSV / ODS)
- Scripts modulares

## Contexto
Proyecto basado en un escenario real de operación logística, adaptado para
fines demostrativos y de portafolio técnico.

## Nota
El código publicado no incluye información sensible, credenciales ni datos reales.
