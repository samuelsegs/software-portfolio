# Mensajes-UnidadesTG

Sistema de automatización desarrollado en Python para el envío de mensajes
informativos a unidades operativas mediante WhatsApp Web.

---

## Descripción

Mensajes-UnidadesTG es una solución de automatización enfocada en optimizar
la comunicación operativa mediante el envío programado de mensajes
informativos, reduciendo tiempos de notificación y errores manuales
en procesos repetitivos.

El sistema toma información estructurada de asignaciones operativas y
genera mensajes claros y estandarizados para su envío automatizado.

---

## Objetivo

- Automatizar la comunicación operativa con unidades
- Reducir tiempos de notificación manual
- Estandarizar el formato de mensajes enviados
- Apoyar la eficiencia en procesos logísticos

---

## Tecnologías utilizadas

- Python
- Automatización de navegador (WhatsApp Web)
- Manejo de archivos (CSV / ODS / XLS)
- Interfaz gráfica de escritorio
- Arquitectura modular de scripts

## Flujo general del sistema

1. Carga de información operativa desde archivos estructurados
2. Procesamiento y validación de datos
3. Generación automática del mensaje
4. Envío del mensaje vía WhatsApp Web
5. Confirmación visual del envío

---

## Contexto del proyecto

Proyecto basado en un escenario real de operación logística,
adaptado para fines demostrativos y de portafolio técnico.

El código publicado ha sido ajustado para eliminar información sensible
y no incluye credenciales, sesiones ni datos reales.
